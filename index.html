INTERNET

Internet adalah jaringan global yang terdiri dari jutaan komputer dan perangkat yang terhubung satu sama lain menggunakan protokol komunikasi standar, yaitu Internet Protocol (IP). Ketika pengguna mengakses informasi atau layanan di internet, data ditransmisikan dalam bentuk paket-paket kecil yang melewati berbagai jaringan hingga mencapai tujuannya.

Secara umum, cara kerja internet melibatkan :

  1. Client (Pengguna): Menggunakan perangkat seperti komputer atau smartphone untuk meminta data.

  2. Server: Komputer yang menyediakan layanan atau data, seperti situs web.

  3. Paket Data: Informasi yang dikirim dalam paket-paket kecil menggunakan TCP/IP (Transmission Control Protocol/Internet Protocol).

  4. Rute Jaringan: Paket data melewati router, switch, dan perangkat jaringan lainnya hingga sampai ke tujuan.

  5. Respons: Server mengirim balasan dalam bentuk paket data kembali ke client.

Source : https://www.cloudflare.com/learning/network-layer/how-does-the-internet-work/

HTTP / HTTPS

Hypertext transfer protocol (HTTP) adalah sebuah protokol atau seperangkat aturan komunikasi untuk komunikasi klien-server. Saat Anda mengunjungi situs web, peramban Anda mengirimkan permintaan HTTP ke server web, yang merespons dengan respons HTTP( HTTP bekerja pada port 80 ). Singkatnya, protokol HTTP adalah teknologi dasar yang mendukung komunikasi jaringan. Seperti namanya, hypertext transfer protocol secure (HTTPS) adalah versi atau perpanjangan HTTP yang lebih aman. HTTPS bekerja pada port 443 dan memberikan keamanan tambahan dengan mengenkripsi data, sehingga melindungi dari serangan peretas.

Cara kerja HTTP :

  1. Browser mengirimkan permintaan HTTP ke server.

  2. Server merespons dengan mengirimkan halaman web atau data yang diminta.

  3. Browser menampilkan data tersebut kepada pengguna.

Cara kerja HTTPS:

  1. Klien dan server memulai koneksi SSL/TLS.

  2. Browser memverifikasi sertifikat keamanan server.

  3. Data yang dikirimkan antara browser dan server dienkripsi untuk memastikan kerahasiaan.

Source : https://www.ssl.com/faqs/what-is-https/

DOMAIN NAME

Domain Name adalah alamat yang digunakan untuk mengidentifikasi suatu situs web di internet. Misalnya, domain "example.com" digunakan untuk mengakses situs web tertentu tanpa harus mengingat alamat IP-nya. Domain name berfungsi sebagai pengganti alamat IP yang sulit diingat dan lebih memudahkan pengguna untuk mengakses layanan di internet.

Nama domain terdiri dari beberapa bagian yang dipisahkan oleh titik :

  1. Top-level domain (TLD): Merupakan bagian akhir dari nama domain, seperti ".com", ".org", atau ".net".

  2. Second-level domain: Bagian utama dari nama domain, seperti "google" dalam "google.com".

  3. Subdomain: Bagian opsional di depan domain utama, seperti "mail" dalam "mail.google.com".

Cara kerja nama domain :

  1. Pengguna mengetik nama domain di browser.

  2. Browser mengirimkan permintaan ke DNS (Domain Name System) untuk menemukan alamat IP dari domain tersebut.

  3. DNS mengembalikan alamat IP terkait.

  4. Browser menggunakan alamat IP untuk menghubungi server dan mengambil konten yang diminta.

Source: https://www.icann.org/resources/pages/what-2012-02-25-en

HOSTING

Hosting adalah layanan penyewaan server yang memungkinkan individu atau organisasi menyimpan file dan data situs web mereka di server, sehingga situs web tersebut dapat diakses melalui internet.

Secara umum, cara kerja hosting melibatkan :

  1. Pengguna: Mengunggah file situs web ke server hosting.

  2. Server Hosting: Server menyimpan file situs dan memastikan situs tersebut tersedia dan dapat diakses oleh pengguna di seluruh dunia.

  3. Domain: Pengguna dapat mengakses situs melalui nama domain yang terhubung dengan server hosting.

Source: https://www.godaddy.com/garage/what-is-web-hosting/

DOMAIN NAME SYSTEM (DNS)

Domain Name System (DNS) adalah sistem yang menerjemahkan nama domain seperti "example.com" menjadi alamat IP yang dapat digunakan oleh komputer untuk menemukan server yang tepat di internet.

Cara kerja DNS melibatkan beberapa langkah berikut:

  1. Pengguna memasukkan nama domain di browser.

  2. Resolver DNS melakukan pencarian di sistem DNS untuk menemukan alamat IP yang terkait dengan nama domain tersebut.

  3. Setelah alamat IP ditemukan, browser menggunakan IP tersebut untuk terhubung ke server yang menyimpan situs web yang diminta.

Source: https://www.cloudflare.com/en-gb/learning/dns/what-is-dns/

BROWSER

Browser adalah perangkat lunak yang memungkinkan pengguna untuk mengakses dan menampilkan informasi di internet. Browser menerjemahkan kode HTML, CSS, dan JavaScript yang ada di server menjadi tampilan grafis yang bisa dimengerti dan digunakan oleh manusia.

Cara kerja browser melibatkan beberapa langkah:

  1. Pengguna memasukkan URL atau melakukan pencarian di browser.

  2. Browser mengirimkan permintaan HTTP atau HTTPS ke server yang menyimpan halaman web.

  3. Server mengirimkan respons dalam bentuk file HTML, CSS, dan JavaScript kepada browser.

  4. Browser menerjemahkan dan menampilkan konten tersebut kepada pengguna.

Source: https://developer.mozilla.org/en-US/docs/Web/Browser
